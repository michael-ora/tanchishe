# 霓虹街機貪吃蛇 (Neon Arcade Snake)

## 📖 簡介
這是一個擁有精美霓虹視覺風格和可愛卡通元素的經典貪吃蛇遊戲網頁版。遊戲結合了復古街機的美感與現代的 SVG 向量圖形設計，帶來流暢且趣味十足的體驗。

## ✨ 特色功能

### 🎨 極致視覺體驗
- **霓虹風格 UI**：全螢幕賽博龐克深色背景，搭配青色與洋紅色的發光特效及毛玻璃面板。
- **卡通 SVG 角色**：
  - 🐍 **萌系蛇頭**：大眼睛卡通蛇頭，眼神靈動，支援隨移動方向自動旋轉。
  - 🍎 **多樣食物**：隨機出現 **紅蘋果、圓滾滾倉鼠、可愛小兔、藍色小鳥** 四種不同的可愛造型。
  - ✨ **精緻紋理**：蛇身不再是單調的色塊，而是帶有半透明斑點紋理的漸變青色身體，從頭到尾透明度自然過渡。

### 🎮 流暢遊戲體驗
- **動態動畫**：食物生成時帶有平滑的彈性縮放動畫，不僅好看，視覺回饋更強。
- **雙端支援**：完美支援 PC 鍵盤操作（方向鍵）與行動端觸控點擊操作。
- **完整系統**：
  - 包含模擬的「使用者註冊/登入」系統。
  - 即時分數更新與本地最高分記錄。
  - 遊戲暫停/繼續功能。
  - 詳細的歷史戰績記錄。

## 🚀 如何運行

本專案是一個純前端靜態網頁應用，無需安裝任何依賴庫。

**方法 1：直接打開**
雙擊根目錄下的 `index.html` 文件，使用現代瀏覽器（Chrome、Edge、Safari 等）打開即可開始游玩。

**方法 2：本地伺服器（推薦）**
為了獲得最佳體驗（避免某些瀏覽器的跨域限制），建議使用本地伺服器運行：
- **Python**: 在專案目錄下運行 `python -m http.server`
- **Node.js**: 使用 `http-server` 或 `live-server`
- **VS Code**: 使用 "Live Server" 外掛右鍵打開 `index.html`

## 🛠️ 技術棧

- **HTML5 Canvas**: 遊戲核心渲染引擎，負責高性能的 2D 繪圖。
- **CSS3**: 實現介面佈局、霓虹光暈特效、玻璃擬態背景及響應式適配。
- **JavaScript (ES6+)**: 
  - 採用 OOP（物件導向）模式封裝 `SnakeGame` 類和 `UIController` 物件。
  - 複雜的 SVG 資源動態加載與 Canvas 渲染邏輯。
- **SVG**: 所有遊戲角色（蛇頭、蘋果、動物）均使用 SVG 代碼內嵌實現，無需外部圖片資源，確保高清無損。
