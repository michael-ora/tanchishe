# 霓虹街机贪吃蛇 (Neon Arcade Snake)

## 📖 简介
这是一个拥有精美霓虹视觉风格和可爱卡通元素的经典贪吃蛇游戏网页版。游戏结合了复古街机的美感与现代的 SVG 矢量图形设计，带来流畅且趣味十足的体验。

## ✨ 特色功能

### 🎨 极致视觉体验
- **霓虹风格 UI**：全屏赛博朋克深色背景，搭配青色与洋红色的发光特效及毛玻璃面板。
- **卡通 SVG 角色**：
  - 🐍 **萌系蛇头**：大眼睛卡通蛇头，眼神灵动，支持随移动方向自动旋转。
  - 🍎 **多样食物**：随机出现 **红苹果、圆滚滚仓鼠、可爱小兔、蓝色小鸟** 四种不同的可爱造型。
  - ✨ **精致纹理**：蛇身不再是单调的色块，而是带有半透明斑点纹理的渐变青色身体，从头到尾透明度自然过渡。

### 🎮 流畅游戏体验
- **动态动画**：食物生成时带有平滑的弹性缩放动画，不仅好看，视觉反馈更强。
- **双端支持**：完美支持 PC 键盘操作（方向键）与移动端触控点击操作。
- **完整系统**：
  - 包含模拟的“用户注册/登录”系统。
  - 实时分数更新与本地最高分记录。
  - 游戏暂停/继续功能。
  - 详细的历史战绩记录。

## 🚀 如何运行

本项目是一个纯前端静态网页应用，无需按照任何复杂的依赖库。

**方法 1：直接打开**
双击根目录下的 `index.html` 文件，使用现代浏览器（Chrome、Edge、Safari 等）打开即可开始游玩。

**方法 2：本地服务器（推荐）**
为了获得最佳体验（避免某些浏览器的跨域限制），建议使用本地服务器运行：
- **Python**: 在项目目录下运行 `python -m http.server`
- **Node.js**: 使用 `http-server` 或 `live-server`
- **VS Code**: 使用 "Live Server" 插件右键打开 `index.html`

## 🛠️ 技术栈

- **HTML5 Canvas**: 游戏核心渲染引擎，负责高性能的 2D 绘图。
- **CSS3**: 实现界面布局、霓虹光晕特效、玻璃拟态背景及响应式适配。
- **JavaScript (ES6+)**: 
  - 采用 OOP（面向对象）模式封装 `SnakeGame` 类和 `UIController` 对象。
  - 复杂的 SVG 资源动态加载与 Canvas 渲染逻辑。
- **SVG**: 所有游戏角色（蛇头、苹果、动物）均使用 SVG 代码内嵌实现，无需外部图片资源，确保高清无损。
